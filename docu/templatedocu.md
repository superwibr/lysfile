Files docu<br/>
	LYS files are built from a template written in json.<br/>
	The LYSfile types are written as abbreviations, all in caps:<br/>
		|	Type					|	Abbreviation	|<br/>
		+---------------------------+-------------------+<br/>
		|	Typeless				|	NTYPE			|<br/>
		|	Bundle					|	BUNDL			|<br/>
		|	Links					|	LINKS			|<br/>
		|	Readable bundle			|	RBUND			|<br/>
		|	Binary JSON template	|	BJTEMP			|<br/>
		+---------------------------+-------------------+<br/>
