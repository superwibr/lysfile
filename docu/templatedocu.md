Files docu
	LYS files are built from a template written in json.
	The LYSfile types are written as abbreviations, all in caps:
		|	Type					|	Abbreviation	|
		+---------------------------+-------------------+
		|	Typeless				|	NTYPE			|
		|	Bundle					|	BUNDL			|
		|	Links					|	LINKS			|
		|	Readable bundle			|	RBUND			|
		|	Binary JSON template	|	BJTEMP			|
		+---------------------------+-------------------+
